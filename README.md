# tubair
Arduino open source code for tubair project

# v0.0.1
## User authentification 
### Step 1 
Create an account on tubair.bitbucket.org and change USERNAME and PASSWORD variable into api.h file. Branch one green led on
GPIO 2 and one red led on GPIO 4. If the green led lights up, the connexion to our server is well initialized, otherwise you
may have set a wrong username or password into the api.h file.
